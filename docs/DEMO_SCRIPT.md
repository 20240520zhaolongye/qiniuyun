# Demo 视频讲解稿

建议视频控制在 3 到 5 分钟，按下面顺序录制。

## 1. 开场

大家好，我的作品是 SpriteForge，一个面向 2D 游戏开发的素材生成工具。它可以通过文本描述和参数配置，生成素材预览、动画帧、Sprite Sheet 和 JSON 元数据。

## 2. 作品完整度

页面左侧是生成参数，包括素材名称、文本描述、素材类型、美术风格、尺寸、视角、动画、帧数、FPS 和导出目标。

点击“随机”可以快速切换示例。项目支持怪物、角色、道具、UI 图标、Tile 地块和技能特效，覆盖了常见 2D 游戏素材需求。

## 3. 核心功能演示

修改描述或参数后点击“生成素材”。中间区域会显示单帧预览和 Sprite Sheet。

点击“播放”可以查看帧动画效果。点击“透明”可以切换透明背景预览。

下方展示结构化 Prompt，后续可以接入 Stable Diffusion、ComfyUI 或其他图像生成接口。

## 4. 导出流程

右侧可以下载单帧 PNG、Sprite Sheet PNG 和 JSON 元数据。JSON 中包含帧位置、尺寸、持续时间、动画名和导出目标，方便 Unity、Godot、Cocos Creator 等游戏引擎读取。

## 5. 开发质量

项目采用 C++ 后端加浏览器前端。C++ 负责本地 HTTP 服务、参数解析、Prompt 构建、素材计划和元数据生成；JavaScript 负责页面交互和 Canvas 渲染。

项目提供 `/api/plan` 生成接口和 `/api/health` 健康检查接口，并通过自动化测试验证前端工具函数、静态资源服务和 API 生成流程。

## 6. 总结

这个作品完成了从参数输入、素材生成、动画预览到资源导出的完整工作流，同时保留了后续接入真实 AI 图像生成服务的扩展空间。
